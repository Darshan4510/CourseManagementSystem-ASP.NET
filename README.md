# Course Management System (ASP.NET Web Forms)

## Project Description
This is a Course Management System developed using ASP.NET Web Forms and SQL Server.

## Features
- User Login & Registration
- Role-based access (Admin / Student)
- Admin can Add, Edit, Delete courses
- Students can view and apply for courses
- Duplicate course application prevention
- Password hashing using SHA256
- ADO.NET for database operations

## Technologies Used
- ASP.NET Web Forms
- C#
- SQL Server
- ADO.NET
- HTML/CSS

## Database Tables
- Users
- Courses
- Applications

## How to Run
1. Open the solution in Visual Studio
2. Update SQL connection string if required
3. Run the project using IIS Express
